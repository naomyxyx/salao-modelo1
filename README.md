## Descrição

Rupi é um site institucional e portfólio digital desenvolvido para apresentar os serviços, a identidade visual e a proposta de atendimento do salão.

A proposta do projeto é comunicar a ideia de que cabelo é uma forma de expressão individual, oferecendo uma experiência visual acolhedora e alinhada aos valores da marca.

## Funcionalidades

- Portfólio interativo com carrossel 3D;
- Galeria adaptada para dispositivos móveis;
- Exibição dinâmica de serviços;
- Animações e microinterações personalizadas;
- Seção de apresentação da equipe;
- Informações de localização e horário de atendimento;
- Integração com Google Maps;
- Layout responsivo.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Swiper.js
